# Godata Tools Webapp

## Requirements
- nodejs v14.5.0 or greater
- [Yarn](https://yarnpkg.com)

## Setup
- Install dependencies: `yarn install`
- If you do not have the backend server running locally, you can modify `.env.development` file to use a deployed server.
To do so, modify the `REACT_APP_API_URL` entry in `.env.development`.
- Start the frontend: `yarn start`. This will make the front end accessible on `http://localhost:3000`.
